XML

1. What is XML?

XML stands for Etensible Markup Language. XML was designed to store and transport data. It was also designed to be self-descriptive. XML can be used for data exchange between different applications and platforms.

2. How is XML used ?

XML is mostly used for data exchange. It uses a simple way of structuring and describing data so that other applications can use it. Many companies use XML to store data, it can easily be exchanged from web server to a client.

JavaScript

1. What is JavaScript ?

JavaScript is a programing language used in web development. It is used to create interactivity to web pages. For example it can manipulate the contents of webpages, repsond to users actions, and validate user input.

2. How is JavaScript related to HTML and CSS ?

JavaScript is related to HTML and CSS because JavaScript is used to add interactivity to the contents of a webpage. HTML is used to structure a webpaged while CSS is used to style it. They all work together to create a user active webpage.
